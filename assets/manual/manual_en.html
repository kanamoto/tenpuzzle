Ten Puzzle

Source code:https://github.com/kanamoto/tenpuzzle/

Rule

"TEN PUZZLE" is a puzzle that uses four single-digit numbers and the four arithmetic operations to make ten.
e.g. 2,4,7,8 => 2 × (8 - 7 + 4)

Four numbers are given at random. All combinations are solvable.
Create an equation so that it is 10.
The numbers and symbols should be arranged so that the equation is from left to right.
When you have an expression that equals 10, it is clear.

0...9
